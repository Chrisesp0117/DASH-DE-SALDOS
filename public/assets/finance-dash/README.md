# FINANCE DASH — Logo

Coloque o arquivo de logo nesta pasta com o nome:

```
logo.png
```

Recomendações:
- Formato PNG (fundo transparente ou preto)
- Tamanho mínimo: 128×128 px (quadrado)
- URL pública após deploy: `/assets/finance-dash/logo.png`

Se o arquivo não existir, a interface exibe o fallback "FD" até o logo ser adicionado.
